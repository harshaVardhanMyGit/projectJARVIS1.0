# Project JARVIS

A sophisticated AI chatbot inspired by JARVIS from Iron Man, powered by OpenAI's GPT-4o-mini.

## Description

This project features an intelligent assistant that emulates JARVIS's refined, formal personality with British elegance and subtle wit. The bot addresses users respectfully and provides precise, helpful responses.

## Features

- JARVIS-inspired personality and responses
- Natural conversation interface
- OpenAI GPT-4o-mini integration
- Comprehensive error handling
- Precise and helpful responses

## Prerequisites

- Python 3.7+
- OpenAI API key

## Installation

1. Clone this repository:
```bash
git clone https://github.com/YOUR_USERNAME/ProjectJarvis.git
cd ProjectJarvis
```

2. Install required packages:
```bash
pip install openai python-dotenv
```

3. Create a `.env` file in the project root:
```
OPENAI_API_KEY1=your_api_key_here
```

## Usage

Run the notebook or execute the code to interact with JARVIS:

```
Sir: What's the weather like today?
JARVIS: [Response from AI]
```

Commands:
- `exit`, `quit`, or `bye` to terminate the session

## Configuration

- Model: `gpt-4o-mini`
- Max tokens: 150
- Temperature: 0.3 (balanced precision and creativity)
